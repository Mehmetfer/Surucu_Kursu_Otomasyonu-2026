# Surucu Kursu Otomasyonu (2026)

MTSK (Motorlu Taşıt Sürücü Kursu) süreçlerini yönetmek için geliştirilmiş otomasyon uygulaması.

## Proje Hakkında

Bu proje, sürücü kursu operasyonlarını tek bir uygulama üzerinden daha düzenli ve hızlı şekilde yürütmeyi hedefler.

## Özellikler

- Kursiyer ve kullanıcı işlemleri
- Raporlama altyapısı
- Lisans/politika yönetimi
- Yazdırma ve PDF süreçleri

## Teknolojiler

- C#
- .NET (Windows Forms)
- SQL tabanlı veri erişimi

## Kurulum

1. Bu depoyu bilgisayarına klonla.
2. Çözümü Visual Studio ile aç (`Kolera_Mtsk.slnx`).
3. Gerekli NuGet paketlerini geri yükle.
4. Veritabanı bağlantı ayarlarını yapılandır.
5. Projeyi derleyip çalıştır.

## Not

Proje aktif geliştirme aşamasındadır. Dokümantasyon zamanla genişletilecektir.
